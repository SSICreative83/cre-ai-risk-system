# AI-Driven Commercial Real Estate Risk Analytics  
### Deep Learning + Graph Neural Network + LLM Integration

## Overview
This project presents an AI-native approach to commercial real estate (CRE) portfolio risk analysis.  
Traditional CRE analytics evaluates assets independently. This system models CRE as a connected network, capturing spatial and economic dependencies across properties, borrowers, and markets.

## Key Features
- **Deep Learning (DNN):** Predicts loan default probability using financial, macroeconomic, and market features  
- **Graph Neural Network (GNN):** Models relationships across properties and borrowers to capture risk propagation  
- **LLM Integration:** Synthesizes market insights from earnings calls and news  
- **Portfolio Analytics Dashboard:** Visualizes risk distribution and emerging trends  

## Architecture
- Data Layer: Property, loan, macroeconomic, and market data  
- Model Layer: DNN + GNN  
- Intelligence Layer: LLM-based text synthesis  
- Visualization Layer: Portfolio risk dashboard  

## Why This Matters
Commercial real estate portfolios are highly interconnected.  
This system captures:
- Geographic dependencies  
- Industry correlations  
- Borrower relationships  

→ Enabling more accurate and forward-looking risk assessment.

## Example Outputs
- Property-level risk scores  
- Portfolio risk heatmap  
- Emerging market risk signals  
- LLM-generated market summaries
